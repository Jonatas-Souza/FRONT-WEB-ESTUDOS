Abaixo um exemplo de lista de definição:

HTML
HyperText Markup Language (Linguagem de marcação de hypertexto). Linguagem de marcação utilizada para criar páginas web.
CSS
Cascading Style Sheets (Folha de estilo em cascata). Linguagem de estilo utilizada para definir a apresentação de documentos HTML.
JavaScript
Linguagem de programação utilizada para adicionar interatividade às páginas web.
